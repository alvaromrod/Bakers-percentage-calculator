# Baker's Percentage Calculator

A simple and efficient Android application for calculating baker's percentages. This app helps bakers scale recipes and understand ingredient ratios with ease.

## Features

*   **Recipe Management:** Create, edit, and delete recipes.
*   **Baker's Percentage Calculation:** Automatically calculates ingredient weights based on flour weight and percentages.
*   **QR Code Sharing:** Share recipes instantly via QR codes.
*   **Backup & Restore:** Save your recipes to a JSON file and restore them later.
*   **Offline First:** All data is stored locally on your device.

## Tech Stack

*   **Language:** Kotlin (Migrated from Java)
*   **UI:** XML Layouts with ViewBinding
*   **Database:** SQLite (via `SQLiteOpenHelper`)
*   **Build System:** Gradle (Kotlin DSL)
*   **Libraries:**
    *   AndroidX (AppCompat, ConstraintLayout, Activity)
    *   Material Components
    *   Gson (JSON parsing)
    *   ZXing (QR Code scanning)
    *   Kotlin Coroutines (Async operations)

## Getting Started

### Prerequisites

*   Android Studio Iguana or later
*   JDK 17

### Building the Project

1.  Clone the repository.
2.  Open the project in Android Studio.
3.  Sync Gradle project.
4.  Run on an emulator or physical device.
